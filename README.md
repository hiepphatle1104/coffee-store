# Coffee Store

## Techstacks

- Docker
- NodeJS
- Golang
- Typescript
- MongoDB

## Installation

- Clone this project

```bash
git clone https://github.com/hiepphatle1104/coffee-store.git

```

- Go to api folder

```bash
docker compose up -d

make run

OR

go run cmd/api/main.go

```

- Go to electron app folder

```bash
npm install && npm run dev

```
